# Bank Marketing Prediction

## Overview
This project predicts whether a customer will subscribe to a bank term deposit using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML
- CSS

## Features
- Data preprocessing
- Machine Learning model training
- Flask web application
- Customer subscription prediction

## Files Included
- Code.ipynb
- app.py
- requirements.txt
- Train.csv
- Test.csv
